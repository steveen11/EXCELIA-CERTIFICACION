# ExcelIA_SAMUEL ARDILA

Proyecto de certificación de Excel con IA basado en un caso académico de ventas mensuales y proyección del próximo trimestre.

## Pregunta clave
¿En qué mes del próximo trimestre se proyectan mayores ventas y qué categoría lidera el acumulado?

## Resultado principal
- Mes proyectado con mayor venta: **Marzo 2025**
- Valor proyectado máximo: **$447,393**
- Categoría líder: **Tecnología**
- Región líder: **Norte**

## Estructura del Excel
1. **Portada**
2. **Datos_Raw**
3. **Datos_Limpios**
4. **Dashboard**
5. **Pronostico**
6. **Resumen**
7. **Documentacion**

## Herramientas y fórmulas destacadas
- `TRIM` y `PROPER` para limpieza de texto.
- `VALUE` e `IFERROR` para estandarización numérica.
- `SUMPRODUCT` y `SUMIF` para consolidar métricas.
- `INDEX + MATCH + MAX` para identificar líderes.
- `AVERAGE` para el modelo simple de proyección.

## Uso de IA
La IA se utilizó como apoyo para:
- definir la pregunta clave,
- estructurar el dashboard,
- redactar conclusiones,
- preparar el guion del video.

## Reflexión ética
La IA fue usada como apoyo y no como sustituto de la validación humana. Las fórmulas y la interpretación final fueron revisadas manualmente.
